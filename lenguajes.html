Lenguajes

Lenguajes de programacion

HTML Y CSS

HTML es un lenguaje de marcado que nos permite indicar la estructura de nuestro documento mediante etiquetas. CSS permite la separación entre la presentación y la estructura. CSS puede definir el color, fuente, alineación del texto, tamaño, bordes, espaciado, capas y muchas otras características tipográficas, y pueden aplicarse distintos estilos de impresión y de pantalla.

CRUD

En informática, CRUD es el acrónimo de "Crear, Leer, Actualizar y Borrar" (del original en inglés: Create, Read, Update and Delete), que se usa para referirse a las funciones básicas en bases de datos o la capa de persistencia en un software.

CHART JS

Chart.js es una biblioteca JavaScript gratuita de código abierto para la visualización de datos, en graficas ya se en tiempo real de una appi o de una base de datos para graficar.

REACT JS

React (también llamada React.js o ReactJS) es una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página. Utilizando la libreria de NodeJS.
